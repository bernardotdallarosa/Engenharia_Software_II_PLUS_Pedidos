# plus-mfe-ped

Microfrontend de **Pedidos** (MFE7). Estrutura mínima para Module Federation; UI de negócio virá após o contrato do `plus-ms-ped`.

## Desenvolvimento

```bash
npm install
npm run dev
```

Standalone: http://localhost:4007  
Remote: http://localhost:4007/assets/remoteEntry.js — expõe `PlaceholderPage`.
